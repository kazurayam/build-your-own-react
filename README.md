## ユニットテストを追加する


『仮想DOM完全理解！』の記事はユニットテストについてなにも書いていない。著者がユニットテストをしなかったはずはない。きっと記事を簡略化するために省略したのだろう。わたしが写経している途中でどうしてもユニットテストを書いて動作を確認したかった。さもないと身をもって理解するのが難しかった。そこでユニットテストを追加した。下記の記事を参考にした。

[（自分の）JavaScriptのユニットテストの書き方,Zenn,mizchi](https://zenn.dev/mizchi/articles/my-test-policy)
